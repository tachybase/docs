# 表格卡片

## 介绍

表格卡片是 Tachybase 内置的核心数据卡片之一，以表格形式展示和管理结构化数据。它具备灵活的配置选项，可定制表格列、列宽、排序规则、数据范围等，同时内置了多种操作供配置：筛选、新建、复制、编辑、删除等。
