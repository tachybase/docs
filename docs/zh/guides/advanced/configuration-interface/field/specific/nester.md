# 子表单

### 概述

子表单适用于先创建关系数据后进行关联的场景。多层级的关系数据通过嵌套表单形式清晰展示。与数据选择器和下拉选择器不同，子表单可以直接在当前页面的卡片中维护关系表字段，且关系数据与主表一起提交。

### 字段配置项
#### 允许添加/关联多条（默认开启）
开启此功能后，用户可以通过点击“+”号添加多条记录。

![](../../../../../../public/nester3.png)


#### 字段组件
支持切换为其他关系[字段组件](../field-settings/field-component.md)，例如下拉选择、数据选择器等。
