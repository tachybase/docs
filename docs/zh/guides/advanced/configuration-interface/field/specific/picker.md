# 数据选择器

### 概述

数据选择器以弹窗表格的形式展示，支持从目标表中选择现有数据进行关联，或为目标表添加数据后进行关联。

![](../../../../../../public/fieldSetting-date-picker1.png)


示例：产品表有多对多关系字段「零件」。

已选中的数据不会再显示在表格选择器中。在表格选择器中，用户还可以对关系目标表进行进一步管理，如新增、删除、导入导出等。
