# 文件管理器

### 概述

文件管理器是用于处理关系目标表为文件表的关系字段组件，便于管理和关联文件数据。

![](../../../../../../public/fieldSetting-file-manager2.png)
![](../../../../../../public/fieldSetting-file-manager1.png)

### 字段配置项

- **[字段组件](../field-settings/field-component.md)**

支持切换为其他关系字段组件，如下拉选择、数据选择器等。

- **快速上传（默认开启）**

允许直接上传文件，并自动关联到当前记录。

- **选择文件（默认开启）**

可从已上传的文件中选择并关联到当前记录。
