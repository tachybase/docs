# 表单筛选卡片

## 介绍

表单筛选卡片可以和数据卡片进行连接，连接之后就可以为数据卡片提供筛选能力。

### 给字段设置默认值

像普通的表单卡片一样，可以为普通字段和关系字段设置默认值。当字段存在默认值时，会在页面首次渲染时自动触发一次筛选操作，以使与其相连接的数据卡片展示相匹配的数据。
