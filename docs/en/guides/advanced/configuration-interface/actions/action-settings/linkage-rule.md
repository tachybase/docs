# 操作的联动规则

## 介绍

操作的联动规则基于上下文数据配置条件和执行结果。通过配置操作的联动规则，控制操作的状态（显示、隐藏、启用、禁用）。

![20240423113057](/actions/linkage-rule.png)

## 使用说明

![20240413102150](/actions/linkage-rule-1.png)

满足条件时（无条件默认通过），触发执行，支持在条件判断中使用常量/变量。

### 哪些按钮可以配置联动规则

目前仅支持有数据上下文的的按钮配置联动规则。

表格、甘特图等卡片的行按钮；

详情卡片的按钮；
